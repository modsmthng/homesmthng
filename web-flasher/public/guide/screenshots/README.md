# Guide screenshots

Drop PNG files into this directory using the names below. The guide shows a
labelled placeholder until a file exists; no HTML or JavaScript changes are
needed.

Hardware photos should be square:

- `hardware-trgb.png`
- `hardware-lilygo-amoled.png`
- `hardware-waveshare-amoled.png`

Display screenshots should be square:

- `display-first-setup.png`
- `display-switches.png`
- `display-b1-clock.png`
- `display-settings-standby.png`

Web UI screenshots work best at a 16:10 aspect ratio:

- `web-overview.png`
- `web-homekit-wifi.png`
- `web-appearance-standby.png`
- `web-time-weather-battery.png`
